## Miről szól ez a dokumentáció?

Ez az oldal az ElyOS platformon dolgozó fejlesztőknek szól — legyen szó a core fejlesztéséről, saját alkalmazások készítéséről vagy plugin fejlesztésről. Megtalálod itt a monorepo felépítését, a SvelteKit routing konvenciókat, az adatbázis-kezelést, az állapotkezelési mintákat, a szerver akciókat, az autentikációt és a többi alapvető fejlesztési témát.

:::note
Az SDK használata és a külső plugin fejlesztés dokumentációja hamarosan elérhető lesz ezen az oldalon.
:::

## Főbb témakörök

- Fejlesztői környezet beállítása és első lépések
- Monorepo struktúra és projekt architektúra
- SvelteKit routing és oldalak
- Svelte 5 runes alapú állapotkezelés
- Server actions – `command` / `query` minta
- Adatbázis – Drizzle ORM, sémák, migrációk
- Autentikáció – better-auth integráció
- Többnyelvűség – adatbázis-alapú i18n
- Beépített alkalmazások fejlesztése
- Naplózás, e-mail, fájlkezelés
- Tesztelés és deployment
