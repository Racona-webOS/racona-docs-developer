## What is this documentation for?

This site is intended for developers building on top of the ElyOS platform. Here you'll find architecture overviews, plugin system internals, SDK reference, server-side APIs, and integration guides.

## Key Topics

- Plugin development and deployment
- WebOS SDK usage
- Server-side APIs and database schema
- Authentication and authorization
- Building custom applications
